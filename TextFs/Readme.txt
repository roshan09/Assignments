Compile : g++ -std=c++11 -o Script Script.cpp
Execute : ./Script

Enter following commands:-

1)create (Create a file) :-
create file_name

2)copy (Copy file from external files to internal files) :-
copy source_file_path destination_file_name

3)list (List all files ):-
ls

4)echo (display contents of  particular file ) :-
echo file_name

5)delete (delete particular file) :-
delete file_name
